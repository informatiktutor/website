# Retrieve Google Forms ratings via Google API

Quickstart: https://developers.google.com/forms/api/quickstart/python?hl=de

Create an `.env` file with the following values:

- FORM_ID: The ID of your form, just copy it from the URL

Get your Google API credentials as described in the documentation above.
Rename the downloaded JSON file to `client_secrets.json`
and move it to the `ratings` directory.
Make sure it is ignored by git (check `.gitignore`).
